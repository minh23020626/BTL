# PonPon
1,Mô Tả: 
Game được xây dựng lại lấy ý tưởng từ game dino của chorm, nhân vật bạn điều khiển là một chú mèo phiêu lưu trên sa mạc và né chướng ngại vật

2,Hướng dẫn cài đặt:
- Cần download sdl 2 và cài đặt theo hướng dẫn https://lazyfoo.net/tutorials/SDL/01_hello_SDL/windows/index.php
- Tiếp theo cài thêm các file ttf, mixer, image và môi trường phù hợp
- Sau đó tải xuống clone source xuống github
- Sử dụng các trình để chạy code và tận hưởng game
